# 毛泽东年谱

## 版本概述

《毛泽东年谱》是由中共中央文献研究室（后改为中共中央党史和文献研究院）编撰的编年体著作，以翔实可靠的历史文献资料为依据，按年、月、日顺序详细记述毛泽东的生平、思想、经历和实践活动。主要版本如下：

### 一、1993年版（三卷本）

- **编撰单位**：中共中央文献研究室
- **出版时间**：1993年8月（纪念毛泽东100周年诞辰）
- **主编**：逄先知、冯蕙，副主编：赵福亭、吴正裕、姚旭、黄允升
- **分卷**：
  - 上卷：1893-1937年6月（新民主主义革命前期）
  - 中卷：1937年7月-1945年8月（抗日战争时期）
  - 下卷：1945年8月-1949年9月（解放战争时期）
- **特点**：首次详细记述毛泽东思想和生平业绩的编年体著作，经过八年精心编撰而成

### 二、2023年版（九卷本）

- **编撰单位**：中共中央党史和文献研究院
- **出版时间**：2023年10月
- **主编**：逄先知、冯蕙，副主编：陈晋等
- **特点**：将1993年版三卷本（1893-1949）与1993年版六卷本（1949-1976）合并修订为一部完整的九卷本年谱
- **修订说明**：对原两部年谱进行了整合修订，统一了编写体例，定名为《毛泽东年谱》，共九卷

### 三、毛泽东军事年谱

- **编撰单位**：中国人民解放军军事科学院
- **篇幅**：892页
- **内容**：专门记述毛泽东军事实践活动和军事思想发展的编年体著作

## 2023年版各卷详细摘要

### 第一卷（1893年12月 - 1937年6月）

**时间范围**：1893年12月26日（诞生）至1937年6月
**对应年龄**：出生至四十四岁
**编写人员**：赵福亭、吴正裕、黄允升、蔡钊珍、田逢禄、张素华、李捷、蒋建农

#### 时代背景

本卷涵盖从清末到全面抗日战争爆发前夕的历史时期。中国经历了辛亥革命、军阀混战、五四运动、中国共产党的创立、第一次国共合作、北伐战争、土地革命战争、红军长征等一系列重大历史事件。

#### 重大事件年表

| 年份 | 年龄 | 重大事件 |
|------|------|----------|
| 1893 | 诞生 | 12月26日出生于湖南省湘潭县韶山冲南岸上屋场，字咏芝，后改为润之 |
| 1902-1909 | 9-16岁 | 在韶山多所私塾读书，读四书五经及古代传奇小说（《水浒传》《三国演义》《隋唐演义》等） |
| 1907-1908 | 14-15岁 | 停学在家务农，读《盛世危言》《校邠庐抗议》等，萌发爱国思想 |
| 1910 | 17岁 | 考入湘乡县立东山高等小学堂，读康梁变法书报，崇拜康有为、梁启超 |
| 1911 | 18岁 | 到长沙求学，首次看到《民立报》，拥护孙中山革命党人；辛亥革命爆发后投笔从戎，参加起义新军当列兵 |
| 1912 | 19岁 | 清帝退位后退出军队，考入湖南全省高等中学校，写《商鞅徙木立信论》；后退学自修，在湖南省立图书馆广泛涉猎西方社科书籍 |
| 1913 | 20岁 | 考入湖南省立第四师范学校预科，记万余言《讲堂录》 |
| 1914 | 21岁 | 编入湖南一师本科第八班，受杨昌济、徐特立等影响深远 |
| 1915 | 22岁 | 反对"二十一条"，写"五月七日，民国奇耻。何以报仇？在我学子！"；驱赶校长张干的学潮 |
| 1917 | 24岁 | 发表《体育之研究》于《新青年》 |
| 1918 | 25岁 | 与蔡和森等发起组织新民学会；毕业于湖南一师 |
| 1919 | 26岁 | 五四运动爆发，在长沙领导反帝反军阀斗争；创办《湘江评论》 |
| 1920 | 27岁 | 在北京、上海活动，读马克思主义著作；在长沙建立共产主义小组 |
| 1921 | 28岁 | 7月出席中国共产党第一次全国代表大会；回湖南建立中共湘区委员会 |
| 1922 | 29岁 | 领导安源路矿工人罢工等工人运动 |
| 1923 | 30岁 | 出席中共三大，当选中央执行委员；参加国共合作 |
| 1924 | 31岁 | 参加国民党一大，当选候补中央执行委员；在国民党上海执行部工作 |
| 1925 | 32岁 | 回韶山开展农民运动；写《中国社会各阶级的分析》 |
| 1926 | 33岁 | 主办第六届农民运动讲习所；《国民革命与农民运动》 |
| 1927 | 34岁 | 发表《湖南农民运动考察报告》；八七会议后领导秋收起义；创建井冈山革命根据地 |
| 1928 | 35岁 | 与朱德会师，成立工农革命军第四军；写《中国的红色政权为什么能够存在？》《井冈山的斗争》 |
| 1929 | 36岁 | 古田会议，写《关于纠正党内的错误思想》 |
| 1930 | 37岁 | 写《星星之火，可以燎原》；《反对本本主义》 |
| 1931 | 38岁 | 中华苏维埃共和国成立，当选中央执行委员会主席 |
| 1932-1934 | 39-41岁 | 遭受"左"倾路线排挤，被剥夺军事指挥权 |
| 1934 | 41岁 | 10月开始长征 |
| 1935 | 42岁 | 遵义会议确立在党中央的领导地位；指挥四渡赤水等战役 |
| 1935-1937 | 42-44岁 | 红军到达陕北，建立抗日民族统一战线；写《实践论》《矛盾论》 |

#### 关键节点

1. **1893年诞生**：毛泽东在湖南湘潭韶山冲出生
2. **1910年出乡关**：离开韶山赴外地求学，立志报国
3. **1918年新民学会**：与蔡和森等创建革命团体
4. **1921年中共一大**：参与创建中国共产党
5. **1927年秋收起义**：开始独立领导武装斗争
6. **1928年井冈山会师**：创建红军和根据地
7. **1935年遵义会议**：确立在党中央的实际领导地位
8. **1937年全面抗战前**：《实践论》《矛盾论》发表，奠定哲学基础

### 第二卷（1937年7月 - 1945年8月）

**时间范围**：1937年7月至1945年8月
**对应年龄**：四十四岁至五十二岁
**编写人员**：冯蕙、姚旭、赵福亭、吴正裕等

#### 时代背景

本卷涵盖整个抗日战争时期。从卢沟桥事变到日本投降，国共两党实现第二次合作，中国共产党领导八路军、新四军在敌后开辟抗日根据地，同时在延安开展整风运动和大规模生产运动。

#### 重大事件年表

| 年份 | 重大事件 |
|------|----------|
| 1937.7 | 卢沟桥事变爆发；毛泽东与中共中央发出抗日通电，推动国共合作 |
| 1937.7-8 | 红军改编为八路军；毛泽东提出独立自主的游击战战略方针 |
| 1937.7 | 写《实践论》；8月写《矛盾论》 |
| 1938 | 发表《论持久战》，系统阐述抗日战争的战略方针 |
| 1939 | 发表《〈共产党人〉发刊词》《中国革命和中国共产党》 |
| 1940 | 发表《新民主主义论》 |
| 1941-1942 | 领导延安整风运动 |
| 1942 | 发表《在延安文艺座谈会上的讲话》 |
| 1943 | 领导大生产运动 |
| 1944 | 组织敌后局部反攻 |
| 1945.4-6 | 中共七大召开，毛泽东思想确立为党的指导思想 |
| 1945.8 | 日本投降，抗日战争胜利 |

#### 关键节点

1. **1937年7月7日卢沟桥事变**：全面抗战开始，国共第二次合作
2. **1938年《论持久战》**：科学预见了抗战三阶段
3. **1940年《新民主主义论》**：系统阐述新民主主义革命理论
4. **1942年延安整风**：确立实事求是的思想路线
5. **1945年中共七大**：毛泽东思想写入党章

---

### 第三卷（1945年8月 - 1949年9月）

**时间范围**：1945年8月至1949年9月
**对应年龄**：五十二岁至五十六岁
**编写人员**：陈廉、李炳华、李捷等

#### 时代背景

本卷涵盖解放战争时期。从日本投降后的国共重庆谈判，到全面内战爆发，再到三大战役（辽沈、淮海、平津）取得决定性胜利，最终建立中华人民共和国。

#### 重大事件年表

| 年份 | 重大事件 |
|------|----------|
| 1945.8 | 日本投降后，毛泽东为中共中央起草方针，部署夺取抗战胜利果实 |
| 1945.8-10 | 重庆谈判，签订"双十协定" |
| 1946.6 | 全面内战爆发 |
| 1947.3 | 撤离延安，转战陕北 |
| 1947 | 发表《目前形势和我们的任务》 |
| 1948.9-1949.1 | 指挥辽沈、淮海、平津三大战役 |
| 1949.3 | 中共七届二中全会，提出"两个务必" |
| 1949.4 | 发布《向全国进军的命令》，百万雄师渡长江 |
| 1949.6 | 发表《论人民民主专政》 |
| 1949.9 | 中国人民政治协商会议第一届全体会议 |

#### 关键节点

1. **1945年重庆谈判**：争取和平民主的尝试
2. **1947年转战陕北**：在极端困难条件下指挥全国战局
3. **1948-1949年三大战役**：歼灭国民党军主力
4. **1949年3月七届二中全会**：提出党的工作重心由乡村转移到城市
5. **1949年渡江战役**：南京解放，国民党政权覆灭

### 第四卷（1949年10月 - 1952年12月）

**时间范围**：1949年10月至1952年12月
**对应年龄**：五十七岁至六十岁
**编写人员**：吴正裕、熊华源、黄允升、许蕾

#### 时代背景

本卷涵盖新中国成立初期。毛泽东担任中央人民政府主席、人民革命军事委员会主席。这一时期主要完成了中国大陆的统一、抗美援朝战争、土地改革和"三反五反"运动。

#### 重大事件年表

| 年份 | 重大事件 |
|------|----------|
| 1949.10.1 | 中华人民共和国成立，毛泽东在天安门宣布"中华人民共和国中央人民政府今天成立了！" |
| 1949.10 | 苏联第一个承认新中国；中苏建交 |
| 1950.2 | 签订《中苏友好同盟互助条约》 |
| 1950.6 | 朝鲜战争爆发 |
| 1950.10 | 做出抗美援朝决策，派遣志愿军入朝作战 |
| 1950-1953 | 全国土地改革 |
| 1951-1952 | "三反""五反"运动 |
| 1952 | 国民经济恢复 |

#### 关键节点

1. **1949年10月1日开国大典**：新中国正式成立
2. **1950年抗美援朝决策**："打得一拳开，免得百拳来"
3. **1950-1952年土地改革**：彻底废除封建土地制度

---

### 第五卷（1953年1月 - 1956年9月）

**时间范围**：1953年1月至1956年9月
**对应年龄**：六十岁至六十三岁
**编写人员**：吴正裕、熊华源、黄允升、刘建平

#### 时代背景

本卷涵盖社会主义改造时期。中国开始实施第一个五年计划，对农业、手工业和资本主义工商业进行社会主义改造，召开第一届全国人民代表大会，制定第一部宪法。

#### 重大事件年表

| 年份 | 重大事件 |
|------|----------|
| 1953 | 第一个五年计划开始；提出过渡时期总路线 |
| 1953.1 | 发出"新三反"指示（反对官僚主义、命令主义、违法乱纪） |
| 1953.7 | 朝鲜停战协定签订 |
| 1954.9 | 第一届全国人民代表大会；通过第一部《宪法》；毛泽东当选国家主席 |
| 1955 | 农业合作化高潮 |
| 1956.4 | 发表《论十大关系》 |
| 1956.9 | 中共八大召开 |

#### 关键节点

1. **1953年过渡时期总路线**：确立社会主义工业化与社会主义改造同时并举的方针
2. **1954年第一部宪法**：确立人民代表大会制度
3. **1956年《论十大关系》**：开始探索中国社会主义建设道路
4. **1956年中共八大**：正确分析国内主要矛盾

---

### 第六卷（1956年10月 - 1959年3月）

**时间范围**：1956年10月至1959年3月
**对应年龄**：六十三岁至六十六岁
**编写人员**：李捷、冯蕙、黄允升、郝首栋

本卷涵盖反右运动、大跃进和人民公社化运动时期。社会主义建设进入高潮，但也出现了急于求成的"左"的倾向。

| 年份 | 重大事件 |
|------|----------|

| 1957.2 | 发表《关于正确处理人民内部矛盾的问题》 |
| 1957.4 | 发起整风运动 |
| 1957.6 | 反右派运动开始 |
| 1957.11 | 率团赴莫斯科参加各国共产党和工人党代表会议 |
| 1958.3 | 成都会议，提出"鼓足干劲、力争上游、多快好省地建设社会主义"总路线 |
| 1958.5 | 中共八大二次会议，通过社会主义建设总路线 |
| 1958.8 | 北戴河会议，通过《关于在农村建立人民公社问题的决议》 |
| 1958 | 大炼钢铁运动 |
| 1958.11-12 | 郑州会议、武昌会议，开始纠"左" |
| 1959 | 继续纠"左"，调整人民公社体制 |

#### 关键节点

1. **1957年《关于正确处理人民内部矛盾的问题》**：创立社会主义社会矛盾学说
2. **1958年大跃进和人民公社化运动**：急躁冒进的"左"倾错误
3. **1958年底开始纠"左"**：认识到"一平二调三收款"等问题的严重性

### 第七卷（1959年4月 - 1961年6月）

**时间范围**：1959年4月至1961年6月
**对应年龄**：六十六岁至六十八岁
**编写人员**：冯蕙、张素华、黄允升等

#### 时代背景

本卷涵盖庐山会议、三年经济困难时期。大跃进的后果全面显现，中苏关系开始破裂，国民经济遭遇严重困难。

#### 重大事件年表

| 年份 | 重大事件 |
|------|----------|
| 1959.4 | 中共八届七中全会，调整钢产量指标；当选全国政协名誉主席 |
| 1959.7-8 | 庐山会议，前期纠"左"，后期因彭德怀上书转向反右 |
| 1959.10 | 国庆十周年 |
| 1960 | 三年经济困难全面显现 |
| 1960.7 | 苏联撤走全部在华专家，中苏关系破裂 |
| 1960冬-1961 | 大兴调查研究之风，开始全面调整国民经济 |

#### 关键节点

1. **1959年庐山会议**：从纠"左"转向反右，使"左"的错误进一步发展
2. **1960年中苏决裂**：国际共运大论战开始
3. **1960-1961年经济困难**：开始重新调查农村实际情况

---

### 第八卷（1961年7月 - 1966年9月）

**时间范围**：1961年7月至1966年9月
**对应年龄**：六十八岁至七十三岁
**编写人员**：陈晋、冯蕙、张素华、唐洲雁

#### 时代背景

本卷涵盖国民经济调整时期、"四清"运动和文化大革命酝酿期。中国从三年困难中逐步恢复，同时中苏论战不断升级，国内阶级斗争理论日益发展。

#### 重大事件年表

| 年份 | 重大事件 |
|------|----------|
| 1961 | 七千人大会前期的调整；制定"农业六十条"等 |
| 1962.1 | 七千人大会，初步总结"大跃进"教训 |
| 1962.9 | 中共八届十中全会，提出"阶级斗争必须年年讲、月月讲、天天讲" |
| 1963 | 开展"四清"运动（社会主义教育运动） |
| 1963-1964 | 中苏大论战"九评" |
| 1964 | 第一颗原子弹爆炸成功 |
| 1965.11 | 姚文元发表《评新编历史剧〈海瑞罢官〉》 |
| 1966.5 | 中共中央发出"五一六通知"，文化大革命正式开始 |
| 1966.8 | 中共八届十一中全会，发表《炮打司令部》大字报 |

#### 关键节点

1. **1962年七千人大会**：民主氛围较大的一次会议
2. **1962年八届十中全会**：阶级斗争扩大化理论
3. **1964年原子弹成功**：打破核垄断
4. **1966年文化大革命发动**：十年动乱开始

---

### 第九卷（1966年10月 - 1976年9月）

**时间范围**：1966年10月至1976年9月
**对应年龄**：七十三岁至八十三岁（逝世）
**编写人员**：陈晋、唐洲雁、沈雁昕、许蕾、王香平

#### 时代背景

本卷涵盖文化大革命主要时期和毛泽东晚年。全国陷入动乱，红卫兵运动、全面夺权、林彪事件、批林批孔、反击右倾翻案风等重大事件相继发生。外交上实现了中美关系正常化和中国恢复在联合国的合法席位。

#### 重大事件年表

| 年份 | 重大事件 |
