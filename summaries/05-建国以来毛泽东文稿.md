# 建国以来毛泽东文稿

> 全书共二十册，由中共中央党史和文献研究院编，中央文献出版社出版（2023年12月公开出版）。时间跨度从1949年9月至1976年7月，收录毛泽东的手稿、指示、批语、电报、讲话提纲、书信、批注、诗词、题词、修改的文件等综合文献。原内部发行版（1987-1998年）为十三册，此次公开出版新增文稿并修订注释，扩充为二十册。每册按写作或发表时间编排，是研究建国以来毛泽东思想和决策过程的核心一手文献。

## 一、新中国建立与全国政权巩固（1949年9月--1950年8月）

**涵盖：第1册至第2册**

### 核心议题

- 中华人民共和国开国大典与中央人民政府的组建运行
- 中国人民政治协商会议的召开与《共同纲领》的制定
- 解放战争最后阶段：向中南、西南、西北进军，解放新疆、广东、广西
- 新中国外交开局：中苏建交与毛泽东访苏、签订《中苏友好同盟互助条约》
- 地方各级政权的建立与各界人民代表会议的推广
- 土地改革的前期准备：征询富农策略、起草土地法
- 军队整编复员与海空军筹建
- 少数民族地区工作（新疆、西藏、内蒙古）

### 关键决策

- **富农策略**：毛泽东反复征询各中央局意见，最终确定"保存富农经济、中立富农"的方针，以利于农业生产和社会稳定
- **中苏结盟**：亲赴莫斯科谈判，签订《中苏友好同盟互助条约》，获得苏联的经济援助和军事支持
- **海空军筹建**：在苏联帮助下组建人民海军和人民空军

### 思想要点

1. **人民民主专政的确立**：从"中国人从此站立起来了"到《共同纲领》，毛泽东为新中国的政治制度框架奠定基础，强调工人阶级领导的、以工农联盟为基础的人民民主专政。
2. **统一战线思想**：高度重视安排民主人士参加各级政权机关，大量书信涉及对黄炎培、程潜、张治中等党外人士的统战工作。
3. **中苏结盟**：建国伊始即谋求与苏联建立全面同盟关系，在军事装备、经济建设和干部培训等方面请求苏联援助。
4. **新区土改的审慎态度**：对富农策略反复征询意见，体现"保存富农经济、中立富农"的谨慎方针，避免影响农业生产。
5. **军事战略与建军思想**：金门失利后的教训总结，海空军筹建，军队整编复员，体现了从战争时期向和平建设的转变。

### 关键决策

- **轮番作战方针**：1951年2月，毛泽东提出志愿军轮番作战方针，以持久战应对美军的火力优势，并命令部队轮番入朝参战
- **镇反运动**：1951年1月提出"打得稳、打得准、打得狠"，5月第三次全国公安会议后收紧政策，规定死刑缓期执行等宽大措施
- **停战谈判**：1951年7月起亲自指挥停战谈判，就议程条款、战俘遣返、中立区等具体问题给李克农发出大量指示电报
- **三反五反**：1951年底发动"三反"（反贪污、反浪费、反官僚主义），1952年1月发动"五反"（反行贿、反偷税漏税、反盗骗国家财产、反偷工减料、反盗窃国家经济情报），在全国开展大规模群众运动

### 思想要点

1. **抗美援朝的战略决断**：毛泽东在朝鲜战争爆发后，经过反复权衡做出出兵决策，认为"参战利益极大，不参战损害极大"，体现了以国家安全为重的战略眼光。
2. **战争与建设并举**：在抗美援朝的同时，推进国内土地改革和经济恢复，提出"边打、边稳、边建"的方针。
3. **土地改革的阶级路线**：依靠贫农、雇农，团结中农，中立富农，有步骤地消灭封建土地制度。
4. **政权巩固中的肃反思想**：镇压反革命运动中强调"稳、准、狠"，同时注意政策界限，将捕人杀人的批准权一律提高一级。
5. **"三反"运动的反腐倡廉思想**：反贪污、反浪费、反官僚主义，强调执政党必须保持廉洁。刘青山、张子善大贪污案在此期间查处。

### 关键决策

- **朝鲜停战签字**：1953年7月亲自指挥停战协定签字的最后安排，多次电报指示李克农关于签字仪式程序、军事分界线等细节，7月27日停战协定正式签字
- **粮食统购统销**：1953年10月决定实行粮食计划收购与计划供应（统购统销），这一制度延续数十年，深刻影响城乡关系
- **过渡时期总路线**：1953年正式提出"一化三改"（逐步实现社会主义工业化，逐步实现对农业、手工业和资本主义工商业的社会主义改造）
- **新三反运动**：1953年1月发起反对官僚主义、反对命令主义、反对违法乱纪的"新三反"运动，着力解决基层政权问题
- **全国财经会议**：1953年8月全国财经会议召开，围绕新税制问题展开争论，批评"公私一律平等"的税制改革
- **对资本主义工商业的改造**：1953年7月、9月先后提出关于国家资本主义经济的重要论述，确定"经过国家资本主义完成由资本主义到社会主义的改造"的方针
- **宪法起草**：亲自主持宪法起草工作（1954年1月起在杭州），1954年9月第一届全国人民代表大会通过《中华人民共和国宪法》
- **高饶事件**：1954年处理高岗、饶漱石反党分裂活动，1月在杭州主持起草《关于增强党的团结的决议》，2月周恩来主持高岗问题座谈会，3月邓小平主持饶漱石问题座谈会，9月高岗自杀后通报苏共

### 思想要点

7. **思想文化批判的开端**：从《红楼梦》研究问题入手，发起对胡适派唯心论的批判，进而扩展为思想领域的全面整肃，为后来的反右派运动埋下伏笔。

### 关键决策

- **"九一三事件"**：1971年9月13日林彪乘飞机出逃坠机身亡，毛泽东亲自处理善后，批林整风运动随之展开
- **中美突破**：1970年会见斯诺释放信号，1971年"乒乓外交"、基辛格秘密访华，1972年尼克松正式访华，发表《中美联合公报》
- **联合国席位**：1971年10月中国恢复在联合国的合法席位，毛泽东亲自审阅代表团发言稿
- **邓小平复出**：1972年批准邓小平恢复工作，1973年恢复副总理职务，1975年主持全面整顿
- **批"四人帮"**：1974-1975年多次批评江青等人，但最终又发动"批邓"运动

### 思想要点

1. **"九一三事件"的巨大冲击**：林彪出逃坠机身亡，对毛泽东的文革理论和政治威望造成重大打击，客观上推动了部分拨乱反正。
2. **中美关系的历史性突破**：从"乒乓外交"到尼克松访华，毛泽东以战略家的视野打开中美关系大门，改变了世界格局。
3. **"三个世界"理论**：将世界划分为三个部分，提出中国属于第三世界，为中国的外交战略提供了理论框架。
4. **邓小平的起落**：在1973-1975年支持邓小平主持全面整顿，推进"四个现代化"，但又发动"批邓"运动将其再次打倒，体现了晚年的矛盾心理。
5. **对限制资产阶级法权的思考**：晚年反复强调"搞社会主义革命不知道资产阶级在哪里，就在共产党内"，发动"理论问题"学习运动，把按劳分配等视为"资产阶级法权"加以批判。
6. **老干部政策的松动**：大量批语涉及为老干部恢复工作、解决生活困难等问题，体现了九一三事件后一定程度的政策调整。

## 全书总结

### 文献特点

1. **文献类型丰富**：涵盖电报、批语、指示、书信、讲话提纲、诗词题词、修改文件等多种形式，全面反映毛泽东的日常决策与思想活动。
2. **时间跨度完整**：从1949年9月政协开幕至1976年7月，完整覆盖毛泽东执政的二十七年。
3. **大量首次公开**：许多电报、批语和书信为此前未曾公开发表的第一手材料，具有极高的史料价值。
4. **注释详尽**：每篇附有注释说明背景、涉及人物和事件，便于理解。
5. **通信数量巨大**：涉及大量给民主人士（黄炎培、张澜、陈叔通等）、亲友（毛宇居、刘松林等）、党内干部的个人书信，反映毛泽东的人际交往和统战思想。

### 历史脉络

全书二十册完整记录了毛泽东从开国奠基到晚年执政的思想轨迹与决策历程。可以概括为以下几条主线：

- **政治主线**：从新民主主义到社会主义改造，从阶级斗争扩大化到文化大革命的发动，最终走向"无产阶级专政下继续革命"的理论与实践。
- **经济主线**：从国民经济恢复到第一个五年计划的成功，从大跃进的冒进到经济调整的务实，始终在计划经济体制内探索发展道路。
- **外交主线**：从"一边倒"联苏到中苏分裂，从"反帝反修"到联美抗苏，体现了大国博弈中的战略灵活性。
- **党建主线**：从执政初期的整党建党，到反右派、反右倾、社教运动，直至文化大革命中"踢开党委闹革命"，党内民主和制度遭到严重破坏。
