# 毛泽东传记类文献总览

## 一、官方版《毛泽东传》（六册）

**基本信息**
- 编纂：中共中央文献研究室
- 主编：逄先知、金冲及
- 出版社：中央文献出版社
- 总页数：约2769页（正文从第1页至第2765页），分六册出版
- 全书共83章，另有后记三篇
- 性质：官方权威传记，基于大量第一手档案文献和当事人回忆

### 第四册（419页）

**覆盖时期**：1956年--1958年（社会主义建设探索初期）

**章节结构**（第53--61章）：
- 第53--54章 《论十大关系》到八大（上/下）
- 第55章 创造一个和平的国际环境

- 第59--60章 发动"大跃进"（上/下）
- 第61章 炮击金门

**核心内容**：
第四册记录毛泽东开始探索中国社会主义建设道路的关键时期。通过系统调研三十四个经济部门形成《论十大关系》，提出"百花齐放、百家争鸣"方针，主持中共八大。同时处理苏共二十大引发的国际共运危机。随后整风运动演变为反右扩大化，发动"大跃进"运动，炮击金门应对国际局势。本册反映了毛泽东从审慎探索到急躁冒进的思想转变。

### 官方版传记的特点

1. **档案权威性**：大量引用中共中央档案、会议记录、电报手稿等第一手材料
2. **叙事全面性**：83章覆盖毛泽东一生（1893--1976），政治、军事、经济、外交、文化各方面均有涉及
3. **史实详实性**：每章均有详细注释，引文出处明确
4. **评价审慎性**：既肯定毛泽东的历史功绩，也不回避其晚年的重大错误
5. **集体研究**：由中共中央文献研究室组织编写，主编逄先知、金冲及均为资深党史专家

## 二、《毛泽东大传》（十卷本）

**基本信息**
- 作者：东方直心（笔名）
- 版本：第四版
- 总页数：4163页
- 总字数：约700万字
- 文体：章回体与编年体相结合
- 性质：民间编撰的超大型综合性传记

**卷册结构与时间覆盖**：

| 卷次 | 卷名 | 起止时间 |
|------|------|----------|
| 第一卷 | 横空出世 | 1893.12 -- 1918.6 |
| 第二卷 | 倚天抽剑 | 1918.6 -- 1927.8.5 |
| 第三卷 | 战地黄花 | 1927.8.7 -- 1935.10 |
| 第四卷 | 风流人物 | 1935.11 -- 1945.8.14 |
| 第五卷 | 谁主沉浮 | 1945.8.14 -- 1949.9 |
| 第六卷 | 五洋捉鳖 | 1949.10.1 -- 1953 |
| 第七卷 | 九天揽月 | 1954 -- 1959.8.16 |
| 第八卷 | 乱云飞渡 | 1959.8.18 -- 1966.5.14 |
| 第九卷 | 玉宇澄清 | 1966.5.16 -- 1968.11 |
| 第十卷 | 只争朝夕 | 1968.12 -- 1976.9.9 |

**各卷卷名取自毛泽东诗词**，如"横空出世"出自《念奴娇-昆仑》，"战地黄花"出自《采桑子-重阳》，"风流人物"出自《沁园春-雪》等，颇具文学意味。

**独特视角与特点**：

1. **篇幅浩大**：全书近700万字，是已知篇幅最长的毛泽东传记，远超官方版《毛泽东传》（约200万字）和其他任何版本
2. **章回体叙事**：采用中国古典章回体小说的叙事方式，每章末尾设有悬念（"欲知后事如何，且看下章分解"），穿插作者"东方翁曰"的评点议论，文学性较强
3. **编年体为纲**：以时间为经、事件为纬，逐日逐月推进叙述，史料密度极大
4. **立场鲜明**：作者对毛泽东持高度肯定和深切同情的立场，在叙述中多有感情色彩浓厚的评价
5. **资料广博**：作者声称广泛收集了包括官方档案、回忆录、地方志、报刊资料等在内的海量材料
6. **细节丰富**：对同一事件往往提供比官方传记更为详尽的细节描写和人物对话

**阅读建议**：
- 欲了解权威史实和官方评价，以官方版《毛泽东传》为主
- 欲获取丰富细节和叙事趣味，可参考《毛泽东大传》
- 欲比较不同视角，可将特里尔版作为西方观点的参照
