# 毛泽东军事文集

> 《毛泽东军事文集》共六卷，由中共中央文献研究室和中国人民解放军军事科学院编辑，1993年军事科学出版社、中央文献出版社出版。全书收录了毛泽东自1927年至1972年间的军事文献，共计1600余篇，全面系统地反映了毛泽东军事思想的形成、发展与实践过程。

---

## 第一卷（1927年8月—1934年9月）

### 时期概述

本卷涵盖土地革命战争前期，从1927年八一南昌起义后到1934年中央红军长征前夕。这一时期是中国工农红军从无到有、从小到大、在艰苦斗争中探索中国革命武装斗争道路的关键阶段。收录文献起于1927年8月，止于1934年9月，共计283篇。

### 历史背景

- 1927年大革命失败后，中国共产党独立领导武装斗争
- 秋收起义与井冈山革命根据地的创建
- 中央苏区的建立与历次反"围剿"作战
- 红军从游击战向运动战的战略转变

### 核心思想

1. **"枪杆子里面出政权"**：明确武装斗争是中国革命的主要形式，提出"须知政权是由枪杆子中取得的"这一著名论断
2. **工农武装割据思想**：在农村建立红色政权，以农村包围城市，走不同于俄国十月革命的中国式革命道路
3. **游击战争基本原则**：总结出"敌进我退，敌驻我扰，敌疲我打，敌退我追"的十六字诀
4. **人民军队建设理论**：强调党对军队的绝对领导，确立"三大纪律八项注意"，提出官兵一致、军民一致、瓦解敌军三大原则
5. **运动战与歼灭战思想**：在反"围剿"作战中逐步形成集中优势兵力、各个歼灭敌人的作战方针

### 重点篇目摘要

| 篇目 | 时间 | 要点 |
|------|------|------|
| 《关于枪杆子里面出政权》 | 1927年8月 | 在八七会议上的发言，提出武装夺取政权的根本路线 |
| 《中国的红色政权为什么能够存在？》 | 1928年10月 | 分析红色政权存在和发展的五个条件 |
| 《井冈山的斗争》 | 1928年11月 | 系统总结井冈山根据地斗争经验 |
| 《关于纠正党内的错误思想》 | 1929年12月 | 古田会议决议，确立红军建设的根本原则 |
| 《星星之火，可以燎原》 | 1930年1月 | 批评悲观思想，预见革命高潮必将到来 |
| 《反对本本主义》 | 1930年5月 | 提出"没有调查，没有发言权"的著名论断 |
| 《中国革命战争的战略问题》 | 1936年12月（本卷收入前期部分） | 系统总结中国革命战争的战略规律 |
| 反"围剿"作战系列电令 | 1930—1934年 | 集中优势兵力、各个歼灭敌人的具体实践 |

### 金句

- **"须知政权是由枪杆子中取得的。"**
- **"星星之火，可以燎原。"**
- **"敌进我退，敌驻我扰，敌疲我打，敌退我追。"**
- **"没有调查，没有发言权。"**
- **"革命不是请客吃饭，不是做文章，不是绘画绣花，不能那样雅致，那样从容不迫、文质彬彬，那样温良恭俭让。"**
- **"我们不但要善于破坏一个旧世界，我们还要善于建设一个新世界。"**
- **"红军的打仗，不是单纯地为了打仗而打仗，而是为了宣传群众、组织群众、武装群众，并帮助群众建设革命政权才去打仗的。"**

## 第二卷（1934年10月—1937年7月）

### 时期概述

本卷涵盖中央红军长征到抗日战争全面爆发前夕。时间跨度从1934年10月中央红军开始长征，到1937年7月抗日战争全面爆发。收录文献约200篇。这是中国革命经历最严峻考验后又重新崛起的伟大转折时期。

### 历史背景

- 中央红军第五次反"围剿"失败，被迫进行战略转移
- 遵义会议确立了毛泽东在党和红军中的领导地位
- 红军长征途中的重大军事决策与战役指挥
- 红军到达陕北后的战略部署与东征、西征
- 抗日民族统一战线的酝酿与形成

### 核心思想

1. **遵义会议的军事路线纠正**：批判"左"倾军事教条主义，恢复灵活机动的战略战术
2. **运动战精髓**：在长征途中通过四渡赤水、巧渡金沙江等经典战例，展现了高度灵活的运动战指挥艺术
3. **统一战线下的军事战略转变**：由国内革命战争向抗日民族解放战争转变
4. **积极防御思想**：反对消极防御，主张在战略防御中实施战役战斗的进攻
5. **持久战思想的萌芽**：开始思考未来抗日战争的持久性质

### 重点篇目摘要

| 篇目 | 时间 | 要点 |
|------|------|------|
| 长征途中系列作战电令 | 1934年10月—1935年10月 | 包含四渡赤水、渡过金沙江、强渡大渡河等重大军事决策 |
| 《直罗镇战役的部署》 | 1935年11月 | 红军到达陕北后的关键一战，粉碎敌人对陕北的围攻 |
| 《关于军事战略问题的报告》 | 1935年12月 | 瓦窑堡会议军事报告，提出东征抗日战略 |
| 《中国革命战争的战略问题》 | 1936年12月 | 系统论述中国革命战争规律，总结十年内战经验 |
| 东征、西征战役系列电令 | 1936年2月—8月 | 扩大陕北根据地，准备抗日力量 |
| 《关于目前军事形势的估计》 | 1937年5月 | 分析全面抗战即将到来的形势 |

### 金句

- **"运动战的基本特点，就在于它的流动性。"**
- **"防御的时候叫做防御，进攻的时候叫做进攻，这不是灵活性的表现，只是僵化的表现。"**
- **"中国是一个很大的国家，只要我们的战略战术正确，革命的力量一定能够逐步发展壮大。"**
- **"我们不能要求事实上的常胜将军，这是从古以来就很少的。我们要求在战争过程中一般地打胜仗的勇敢而明智的将军。"**
- **"我们的战略是'以一当十'，我们的战术是'以十当一'，这是我们制胜敌人的根本法则之一。"**

## 第三卷（1937年7月—1940年5月）

### 时期概述

本卷涵盖抗日战争初期至相持阶段前期。时间从1937年7月全面抗战爆发到1940年5月。收录文献约260篇。这一时期，毛泽东深刻分析了抗日战争的性质、特点和规律，提出了全面抗战路线和持久战的战略总方针。

### 历史背景

- 七七事变，全面抗战爆发
- 国共第二次合作，红军改编为八路军、新四军
- 太原会战、徐州会战、武汉会战等正面战场作战
- 八路军深入敌后，开辟抗日根据地
- 抗战进入相持阶段，日军对敌后根据地进行大规模"扫荡"

### 核心思想

1. **持久战理论**：系统论证抗日战争是持久战，最后胜利属于中国，批判亡国论和速胜论
2. **抗日游击战争的战略地位**：将游击战从战术层面提升到战略层面，这是毛泽东军事思想的重大创新
3. **独立自主的山地游击战方针**：在统一战线中保持共产党军队的独立自主
4. **人民战争思想**：动员和依靠群众，建立广泛的抗日民族统一战线
5. **建立抗日根据地**：在敌后广泛建立根据地，形成持久抗战的战略支撑

### 重点篇目摘要

| 篇目 | 时间 | 要点 |
|------|------|------|
| 《反对日本进攻的方针、办法和前途》 | 1937年7月 | 提出全面抗战路线，反对片面抗战 |
| 《抗日游击战争的战略问题》 | 1938年5月 | 系统论述游击战争在抗战中的战略地位和作用 |
| 《论持久战》 | 1938年5月 | 全面系统论述持久战的战略总方针，是毛泽东最重要的军事理论著作之一 |
| 《论新阶段》 | 1938年10月 | 分析抗战进入相持阶段的新形势与新任务 |
| 《战争和战略问题》 | 1938年11月 | 阐明战争与战略在中国革命中的首要地位 |
| 《八路军军政杂志发刊词》 | 1939年1月 | 总结八路军抗战经验 |
| 《抗战十五个月的总结》 | 1938年10月 | 系统总结全面抗战以来十五个月的经验教训 |
| 抗日根据地建设系列指示 | 1937—1940年 | 关于敌后根据地军事、政治、经济建设的全面部署 |

### 金句

- **"战争的伟力之最深厚的根源，存在于民众之中。"**
- **"兵民是胜利之本。"**
- **"武器是战争的重要的因素，但不是决定的因素，决定的因素是人不是物。"**
- **"亡国论者看敌人如神物，看自己如草芥；速胜论者看敌人如草芥，看自己如神物，这些都是错误的。"**
- **"抗日战争是持久战，最后胜利是中国的——这就是我们的结论。"**
- **"游击战争是什么呢？它就是在落后的国家中，在半殖民地的大国中，在长的时期内，人民武装队伍为了战胜武装的敌人、创造自己的阵地所必须依靠的因而也是最好的斗争形式。"**
- **"革命战争是群众的战争，只有动员群众才能进行战争，只有依靠群众才能进行战争。"**

## 第四卷（1940年6月—1943年3月）

### 时期概述

本卷涵盖抗日战争相持阶段中期。时间从1940年6月到1943年3月。收录文献约260篇。这一时期是抗日战争最困难的阶段，日军疯狂"扫荡"敌后根据地，国民党顽固派不断制造摩擦，根据地面临严重困难。

### 历史背景

- 百团大战（1940年8月—12月）
- 皖南事变（1941年1月）
- 日军推行"治安强化运动"，对根据地进行残酷"扫荡"
- 根据地开展精兵简政、大生产运动
- 整风运动的开展

### 核心思想

1. **在困难中坚持和发展**：面对日寇"扫荡"和国民党摩擦的双重压力，提出在斗争中求生存、求发展
2. **精兵简政与自力更生**：通过精兵简政和大生产运动克服物资困难
3. **反"扫荡"作战方针**：发展出"敌进我进"、内线作战与外线作战相结合等灵活战术
4. **统一战线中的斗争策略**：在维护统一战线大局的同时坚持有理、有利、有节的斗争原则
5. **人民战争进一步深化**：完善了武装群众与正规军配合作战的理论

### 重点篇目摘要

| 篇目 | 时间 | 要点 |
|------|------|------|
| 百团大战相关电令 | 1940年8月 | 关于百团大战的部署与指挥 |
| 《关于皖南事变的声明》 | 1941年1月 | 揭露国民党顽固派制造的皖南事变 |
| 反"扫荡"作战系列指示 | 1940—1943年 | 针对日军"扫荡"的作战方针和战术指导 |
| 《关于精兵简政的指示》 | 1941—1942年 | 在困难时期实行精兵简政，提高部队战斗力 |
| 《关于发展生产的指示》 | 1941—1943年 | 开展大生产运动，实现自给自足 |
| 《关于军队政治工作的指示》 | 1942年 | 在困难环境中加强军队思想政治工作 |
| 对各抗日根据地的军事部署 | 1940—1943年 | 华北、华中、华南各根据地的战略协调 |

### 金句

- **"自己动手，丰衣足食。"**
- **"我们的方针是：人不犯我，我不犯人；人若犯我，我必犯人。"**
- **"在战略上我们要藐视一切敌人，在战术上我们要重视一切敌人。"**
- **"困难并不是不可克服的。任何困难，只要我们认识它，正视它，就一定能够找到克服它的办法。"**
- **"团结、进步、抗战，这三件事是不可分离的。"**

## 第五卷（1943年4月—1945年8月）

### 时期概述

本卷涵盖抗日战争相持阶段后期到抗战胜利。时间从1943年4月到1945年8月日本投降。收录文献约250篇。这一时期，敌后根据地度过最困难阶段，开始局部反攻，最终迎来抗日战争的伟大胜利。

### 历史背景

- 抗日根据地进入恢复和再发展阶段
- 解放区开始局部反攻
- 国共谈判与联合政府问题的讨论
- 中共七大召开（1945年4—6月）
- 日本投降，抗战胜利（1945年8月）

### 核心思想

1. **向局部反攻转变**：随着国际形势变化和根据地力量壮大，指导八路军新四军由战略防御转向局部反攻
2. **大反攻的准备**：预见抗战最后胜利的到来，部署大反攻的军事准备
3. **军队正规化建设思想**：提出将游击军转变为正规军，由游击战向运动战转变
4. **军民融合与根据地建设**：进一步深化人民战争的理论与实践
5. **关于未来斗争的前瞻思考**：预见到抗战胜利后可能面临的国内斗争形势

### 重点篇目摘要

| 篇目 | 时间 | 要点 |
|------|------|------|
| 局部反攻作战系列指示 | 1943—1945年 | 部署各根据地的攻势作战 |
| 《论联合政府》 | 1945年4月 | 在中共七大上的政治报告，涉及军事方面的战略部署 |
| 《关于军事路线问题》 | 1945年 | 系统总结抗战时期的军事路线 |
| 大反攻部署电令 | 1945年8月 | 日本投降前后关于受降和军事部署的指示 |
| 《关于目前形势和任务的指示》 | 1945年8月 | 抗战胜利后的形势分析与任务部署 |
| 建立东北根据地相关指示 | 1945年8月 | 派遣部队进入东北的战略部署 |

### 金句

- **"全党团结起来，胜利一定是我们的。"**
- **"军队向前进，生产长一寸，加强纪律性，革命无不胜。"**
- **"应该统一的，必须统一，决不许可各自为政；但是，应该发挥地方的积极性，反对那种以'统一'为名，实际上限制和妨碍地方积极性的做法。"**
- **"只要我们依靠人民，坚决地相信人民群众的创造力是无穷无尽的，因而信任人民，和人民打成一片，那就任何困难也能克服，任何敌人也不能压倒我们，而只会被我们所压倒。"**

## 第六卷（1945年8月—1949年9月）

### 时期概述

本卷涵盖解放战争（第三次国内革命战争）全时期。时间从1945年8月抗日战争胜利到1949年9月新中国成立前夕。收录文献约380篇，是六卷中文献最多的一卷。这一时期是毛泽东军事指挥艺术的巅峰，人民解放军从战略防御到战略进攻再到战略决战，最终取得全国胜利。

### 历史背景

- 重庆谈判与政治协商会议
- 国民党发动全面内战（1946年6月）
- 解放军战略防御阶段（1946年6月—1947年6月）
- 刘邓大军挺进大别山，揭起战略进攻序幕（1947年6月）
- 三大战役：辽沈战役、淮海战役、平津战役（1948年9月—1949年1月）
- 渡江战役与解放南京（1949年4月）
- 解放军向全国进军

### 核心思想

1. **战略上藐视敌人、战术上重视敌人**：面对国民党军暂时的数量和装备优势，坚定必胜信心
2. **集中优势兵力、各个歼灭敌人**：这是毛泽东作战方针的核心原则
3. **十大军事原则**：在1947年12月会议上系统提出人民解放军的十大军事原则，标志着毛泽东军事思想的成熟与系统化
4. **大兵团作战指挥艺术**：三大战役的指挥展现了毛泽东高超的战略决战能力
5. **将革命进行到底**：反对中途妥协，坚决彻底地推翻国民党反动统治
6. **军事斗争与政治斗争相结合**：在军事打击的同时注重政治瓦解

### 重点篇目摘要

| 篇目 | 时间 | 要点 |

| 《关于重庆谈判的指示》 | 1945年8月 | 以谈对谈、以打对打的方针 |
| 《以自卫战争粉碎蒋介石的进攻》 | 1946年7月 | 提出以运动战消灭敌人有生力量 |
| 《集中优势兵力，各个歼灭敌人》 | 1946年9月 | 系统阐述这一核心作战原则 |
| 《三个月总结》 | 1946年10月 | 总结解放战争头三个月的经验，指出今后作战方针 |
| 《目前形势和我们的任务》 | 1947年12月 | 提出"十大军事原则" |

## 全书总结：毛泽东军事思想体系

### 一、根本原则

| 原则 | 内涵 |
|------|------|
| 党指挥枪 | 坚持中国共产党对军队的绝对领导 |
| 人民战争 | 动员和依靠广大人民群众进行战争 |
| 实事求是 | 从中国革命战争的实际出发制定战略战术 |

### 二、战略战术体系

1. **战争观**：战争是政治的继续，革命战争是推动历史进步的手段
2. **人民战争**：战争的伟力之最深厚的根源存在于民众之中
3. **持久战**：在敌强我弱的条件下，通过持久作战消耗敌人，壮大自己
4. **游击战上升到战略地位**：在广大国土上以游击战争消耗和疲惫敌人
5. **运动战**：集中优势兵力，在运动中寻找和创造战机，各个歼灭敌人
6. **阵地战**：在条件成熟时攻占敌人大中城市和坚固据点
7. **歼灭战**：以消灭敌人有生力量为主要目标，不以保守或夺取地方为主要目标

### 三、发展阶段

| 阶段 | 时期 | 主要战法 |
|------|------|----------|
| 土地革命战争 | 1927—1937年 | 游击战为主，后期发展为运动战 |
| 抗日战争 | 1937—1945年 | 持久战中游击战升到战略地位 |
| 解放战争 | 1945—1949年 | 运动战与阵地战结合，大兵团歼灭战 |

### 四、历史意义

毛泽东军事思想是马克思主义军事理论与中国革命战争实践相结合的伟大成果。它不仅指导中国人民取得了革命战争的伟大胜利，也为世界上被压迫民族和被压迫人民的解放斗争提供了宝贵的理论武器。其核心精华——人民战争思想、积极防御战略、灵活机动的战略战术——至今仍具有重要的理论价值和现实意义。
